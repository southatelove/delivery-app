# React Delivery-App

Панель для возможности заказов еды.

## Стек:

- **Typescript**
- **React**
- **Redux-Toolkit**
- **React-Router**
- **React-Hooks**
- **Vite**
- **ModuleCSS && CSS && ClassNames**
- **Figma/Pixso**

## To view the project

Для старта проекта используйте следующие команды: git clone, npm i , npm run dev
